## 3. 主营业务（三大板块）

**门槛**：除"服务代理商"渠道外，所有业务均要求 **VIP Club 会员**。

### 板块一：开源免费版

- 任何人都可以用
- 完全免费

### 板块二：VIP Club 会员

**基础套餐**：
- 价格：**168 元/年**
- 权益：当前 VIP 168 全部既有权益 + **会员有效期内免费使用官方中转服务**

**重要备注**：
- 单纯购买 VIP Club **不需要注册账号**
- 如需使用中转服务，必须去官网注册（支持邮箱/手机号）

**VIP Club 专享增值服务**：

| 服务 | 价格 | 说明 |
|------|------|------|
| 远程安装部署 | 200 元 / 次 | 一次性 |
| 远程技术支持 | 100 元 / 小时 | 含远程操作电脑 + 语音答疑 |

### 板块三：定制开发（仅 VIP Club 会员专享）

**前置条件**：客户必须先成为 VIP Club 会员。

#### 3a. 纯定制开发
- 客户付开发费，我们做交付
- **流程**：VIP 会员 → 确定需求 / 报价 / 交付日期 → 签合同 + 预付 **30%** → 项目执行

#### 3b. OEM 合作
- 客户需有特定领域客群的分发能力
- 我们出 OEM 版（产品名 / 标识换客户）+ 按客群加技能
- 不收开发费，纯分润
- 客户预缴保证金（意向金），分润先从保证金扣
- **保证金金额**：商议而定，目前没有固定标准
- 约定合作期内累计分润 < 保证金 → 保证金不退

**特殊条款**：

| 情形 | 处理 |
|------|------|
| 开发中遇技术问题无法交付 | 全额退意向金 |
| 我们想延期客户不同意 | 全额退意向金 |
| 交付完成后 | 意向金不退 |
| 未征得同意延期交付 | 按意向金 **1%/天** 赔偿 |
| 交付 6 月内分润 < 意向金 | 意向金转为开发补偿，**第 7 个月起**按约定分润 |
| 交付 6 月内累计分润 ≥ 意向金 | 超额之日起**次日**正式分润 |

**流程**：沟通接洽 → 合作方案确定 → 签合同 + 缴意向金 → 开发与交付 → 按月分润

## 4. 渠道 / 合作模式：服务代理商（独立合作模式，不属于主营业务）

**代理商资格**：
- ❌ **不需要**是 VIP Club
- ✅ 必须能统一提供：**硬件 + 算力 + 运维支持服务**

**代理商业务模式**：
- 代理商自购/提供硬件 + 算力 + 运维
- 把 wiseflow 部署到自己的基础设施
- 以 SaaS 模式服务**自己的客户**（代理商 ↔ 客户关系与我们无关）

**我们向代理商提供**：

| 项目 | 计费方式 | 内容 |
|------|----------|------|
| 一次性技术服务费 | **50% 预付 + 50% 交付后付** | OEM + 第一次统一部署 + 培训 + 客户要求的特定功能定制开发 |
| 月度授权费 | 按月支付 | 金额根据具体合作模式另定 |

**流程**：
1. 沟通接洽
2. 合作方案确定
3. 签合同 + 缴纳技术服务费 **50%**
4. 开发与交付
5. 缴纳技术服务费 **剩余 50%**
6. 按月支付授权费
